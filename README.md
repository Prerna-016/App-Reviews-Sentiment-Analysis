
# 📱 App Reviews Sentiment Analysis

A Machine Learning and Natural Language Processing (NLP) project that analyzes user reviews from applications and predicts sentiment as **Positive**, **Negative**, or **Neutral** using review text data.

This project uses a dataset of LinkedIn app reviews (`linkedin-reviews.csv`) for sentiment analysis and customer feedback insights.

---

## 🚀 Features

- Data Cleaning & Preprocessing
- Natural Language Processing (NLP)
- Sentiment Analysis on App Reviews
- Text Vectorization using TF-IDF / CountVectorizer
- Machine Learning Model Training
- Model Evaluation & Accuracy Analysis
- Data Visualization

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- NLTK

---

## 📂 Dataset Used

Dataset File:

```bash
linkedin-reviews.csv
````

The dataset contains user reviews collected from the LinkedIn application and is used to train and test the sentiment analysis model.
